## Installation

### BetterDiscord
1. Download [QuestAutoComplete.plugin.js](https://github.com/Manolo-Kat/Discord_Quest_Auto-Complete/releases/download/Beta/QuestAutoComplete.plugin.js)
2. Place in `%appdata%/BetterDiscord/plugins/`
3. Enable in User Settings â†’ Plugins

### Vencord
1. Download [QuestAutoComplete.tsx](https://github.com/Manolo-Kat/Discord_Quest_Auto-Complete/releases/download/Beta/QuestAutoComplete.tsx)
2. Place in Vencord plugins folder
3. Enable in settings

### Tampermonkey
1. Install from [Greasyfork](https://greasyfork.org/en/scripts/562155-discord-quest-auto-complete)
2. Or manually install from [Here](https://github.com/Manolo-Kat/Discord_Quest_Auto-Complete/releases/download/Beta/QuestAutoComplete.user.js)
## Usage
Click the blue "Start Completing Quests" button in Discord!
