# Discord Quest Auto-Complete

Auto-complete Discord quests with one click!

## Features
- ✅ One-click quest completion
- ✅ Supports all quest types (Video, Play, Stream, Activity)
- ✅ Works on BetterDiscord, Vencord, and Tampermonkey

## Installation

### BetterDiscord
1. Download [QuestAutoComplete.plugin.js](link)
2. Place in `%appdata%/BetterDiscord/plugins/`
3. Enable in User Settings → Plugins

### Vencord
1. Download [QuestAutoComplete.tsx](link)
2. Place in Vencord plugins folder
3. Enable in settings

### Tampermonkey
1. Install from [Greasyfork](https://greasyfork.org/en/scripts/562155)
2. Or manually install from [GitHub](link)

## Usage
Click the blue "Start Completing Quests" button in Discord!
